# Breather

## Installation

1. Clone le repo
2. Ouvre dans Xcode
3. Balance sur ton iPhone (brancher le bigo, le choisir dans product -> destination, cmd+r)
4. Configure l'automatisation dans Raccourcis :
   - Automatisation → App → Instagram → "est ouverte"
   - Ajoute l'action "Breathing Exercise"
   - Choisis l'app
   - Désactive "Demander avant d'exécuter"
